# Milestones

The following major milestones define the structure of the VESTAFI MVP:

- **Authentication Module**
  (Handles OTP-based login and session management)

- **User Profile Module**
  (First-time setup and profile editing features)

- **Admin Management Module**
  (Admin actions including property updates, rent tracking, and withdrawal handling)

- **Property Exploration Module**
  (Property listing, detail view, and investment initiation)

- **Investor Dashboard Module**
  (Displays investments, returns, vault balance, and withdrawal actions)

- **Support Module**
  (Contact support functionality)
